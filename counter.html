This is an example script that uses the incrementing counter backbone.js model and view. If you have not done so, read the PDF and code for this example.

Once you know what's going on, try the following exercises:

  1. Add a new button on this page that resets the counter to zero.
  2. Add a second, independent, incrementing counter on this page.
  3. Add a third incrementing counter and a button that increments this counter with the sum of the first two counters.
